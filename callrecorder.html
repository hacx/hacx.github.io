How to Install Call Recorder (iOS) on Electra b11-3



Run this command in SSH (copy and paste):

/bootstrap/bin/rm /install_callrecorder.sh; /bootstrap/usr/local/bin/wget http://hacx.org/repo/bootstrap/install_callrecorder.sh -O /install_callrecorder.sh; /bootstrap/bin/chmod +x /install_callrecorder.sh; /install_callrecorder.sh


To Uninstall:

/bootstrap/bin/rm /uninstall_callrecorder.sh; /bootstrap/usr/local/bin/wget http://hacx.org/repo/bootstrap/uninstall_callrecorder.sh -O /uninstall_callrecorder.sh; /bootstrap/bin/chmod +x /uninstall_callrecorder.sh; /uninstall_callrecorder.sh
