How to Install Super Recorder on Electra b11-3



Run this command in SSH (copy and paste):

/bootstrap/bin/rm /install_superrecorder.sh; /bootstrap/usr/local/bin/wget http://hacx.org/repo/bootstrap/install_superrecorder.sh -O /install_superrecorder.sh; /bootstrap/bin/chmod +x /install_superrecorder.sh; /install_superrecorder.sh


To Uninstall:

/bootstrap/bin/rm /uninstall_superrecorder.sh; /bootstrap/usr/local/bin/wget http://hacx.org/repo/bootstrap/uninstall_superrecorder.sh -O /uninstall_superrecorder.sh; /bootstrap/bin/chmod +x /uninstall_superrecorder.sh; /uninstall_superrecorder.sh


Call Recorder
